# Biblioteksstyringssystem

Dette er et bibliotekssystem skrevet i Python, der bruger Objektorienteret Programmering (OOP).

## Krav

- VScode eller Python 3.6

Tjek din Python-version med:
```bash
python --version
```

## Sådan starter du programmet

1. Sørg for at filen ligger i den ønskede mappe du vil køre programmet fra:
```
Documents/Projects
└── Bibliotekssystem.py
```

2. Åbn terminalen (CMD) og naviger til mappen og kør:
```bash
python bibliotekssystem.py
```

## Funktioner

Når programmet kører, har den følgende funktioner

* **Bogstyring:** Tilføj, fjern, opdater og søg efter bøger.
* **Medlemsstyring:** Opret og slet medlemmer.
* **Udlånssystem:** Håndtering af udlån og afleveringer med lagerstyring.

## Sådan kører du programmet
1. Åbn terminalen.
